

# Echo VAE

Bandwidth‑Efficient Cardiac Ultrasound Keypoint Compression via Continuous Latent Representation

## Overview

This repository contains the official implementation of **Echo VAE**, a framework for compressing left ventricular keypoints in cardiac ultrasound to enable tele‑echocardiography under extreme bandwidth constraints (<64 kbps). The method encodes 40 anatomical keypoints into a 16‑dimensional latent space using a variational autoencoder with dimension‑wise KL lower‑bounding.
